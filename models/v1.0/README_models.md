# Model Registry v1.0

## Состав артефактов

| Файл | Описание |
|------|----------|
| `classifier.pkl` | Обученный классификатор (Logistic Regression) |
| `vectorizer.pkl` | TF-IDF векторizer |
| `label_encoder.pkl` | Кодировщик категорий |
| `config.json` | Метаданные модели (классы, метрики, дата обучения) |
| `preprocessing_metadata.json` | Параметры предобработки текста |

## Важно

Артефакты модели (`.pkl`) **не включены** в репозиторий — они обучались на продуктовых данных и исключены по соображениям конфиденциальности.

Для получения модели:

1. Подготовьте обучающий датасет (CSV с текстами и категориями)
2. Обучите модель по [quick-start-guide.md](../../quick-start-guide.md) или ноутбукам в `notebooks/`
3. Сохраните файлы в этот каталог

После обучения ML Service автоматически загрузит модель из `models/v1.0/` при старте.
